position is fixed means all the elements are ignoring that fixed element in the regular flow of the webpage.

relative means that the element is places relative to its position of the webpage. Other elements still believe that it is at the same place. It also preserve its position on the webpage.

absolute means we are placing it wrt to the first static element. But in our case there is none so it is doing relative.

To decide which element comes in the top of which element we can use z-index property.

elements with higher z-index will show up on the top. The element will be on the top which have z-index rather than pople who have no z-index.

Here top means sabse peeche.

NOTE : Elements will show on the tpp of their z-value os positive. If an element is not given a z-value, then it will inherit from its parent, and by default the parent's z-value is 0. In our example elements with no-specified z-value have then a z-index of 0 and element with a positive z-index will show on the top of them.
